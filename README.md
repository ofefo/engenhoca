# Engenhoca

A prototype for a improvising machine made in Pure Data.
WIP

## Requirements
You should be able to get most libraries needed using Deken (Help > Find externals) directly on PD.
* freeverb~
* else
* timbreID
* mjlib
* list-abs
* cyclone
* [kmeans](https://github.com/AdrianoMonteiro/kmeans-PD)
